# Images Directory

This directory contains all local images for the Arash Nejad website.

## Directory Structure

```
/public/images/
├── hero/           # Hero section images (main portrait, etc.)
├── about/          # About page images (education logos, hobby photos)
├── books/          # Book cover images
├── quotes/         # Quote author headshots
└── seo/            # SEO images (OG image, video thumbnail)
```

## Image Guidelines

- All images should be optimized for web
- Use WebP format when possible for best compression
- Portrait photos should maintain aspect ratio consistency
- Logo images should be high resolution for clarity
